# Frontend

这里是 Student Planner 的前端工程，技术栈为 React 18 + TypeScript + Vite。

如果你是第一次打开这个仓库，建议先看：

- [仓库首页 README](../../README.md)
- [student-planner/README.md](../README.md)

本目录主要包含：

- `src/pages`: 聊天、日历、课程、通知等页面
- `src/stores`: Zustand 状态管理
- `src/api`: 前端 API 封装
- `src/sw.ts`: PWA / service worker 相关逻辑

常用命令：

```bash
npm install
npm run dev
npm run test
npm run build
```
